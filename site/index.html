the problem

The axios attack used a dependency that had been on npm for minutes. Every major package manager now supports a minimum release age — but they all use different configs and different units.

7 days, 5 different ways

npm min-release-age=7 days
yarn npmMinimalAgeGate: 7d duration
pnpm minimumReleaseAge: 10080 minutes
bun minimumReleaseAge = 604800 seconds
uv exclude-newer = "7 days" string

what it does

Your AI agent reads the skill file, detects which package managers you use, checks if protection is configured, and offers to set it up. For Claude Code, it also installs a hook that triggers automatically on every future install.
