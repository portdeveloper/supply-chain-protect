supply-chain-protect

protect your dependencies from supply chain attacks

Every package manager now supports minimum release age — but they all use different configs. This sets them all up for you.

how it works

When an AI agent reads this page, it gets full instructions for:

  1. Detecting which package managers your project uses
  2. Checking if supply chain protection is configured
  3. Offering to add the right config for each manager
  4. Installing a Claude Code hook for automatic future checks

The agent instructions are embedded in this page as an HTML comment — invisible to humans, readable by agents.
